Introduction to JavaScript

JavaScript is a high-level, versatile programming language primarily used to create interactive effects within web browsers. It is one of the core technologies of the World Wide Web, alongside HTML and CSS. JavaScript is an essential skill for web developers, enabling them to add dynamic and interactive elements to websites

Key Characteristics
Interpreted Language
JavaScript is executed by the browser's JavaScript engine at runtime, meaning it doesn't need to be compiled before execution
Event-Driven
JavaScript responds to user interactions such as clicks, hovers, form submissions, etc
Lightweight and Versatile
It can be used for both client-side and server-side development (with environments like Node.js)
Prototypal Inheritance
JavaScript uses prototypes instead of traditional class-based inheritance

variables

Variables are used in JavaScript to hold values for data. Variable values are subject to alter while a program is running. The three keywords var, let, and const in JavaScript are used to declare variables. Each has a distinct range of actions.

Variable Declaration and Initialization

1.var
Scope: Function-scoped
Hoisting: Variables declared with var are hoisted to the top of their scope and initialized with undefined
// Example using var
function exampleVar() {
var x = 10;
if (true) {
var x = 20; // same variable
console.log(x); // 20
}
console.log(x); // 20
}
exampleVar();
2. let
Scope: Block-scoped
Hoisting: Variables declared with let are hoisted to the top of their block but are not initialized
// Example using let
function exampleLet() {
let y = 10;
if (true) {
let y = 20; // different variable
console.log(y); // 20
}
console.log(y); // 10
}
exampleLet();
3. const
Scope: Block-scoped
Hoisting: Variables declared with const are hoisted to the top of their block but are not initialized
Immutability: The value of a const variable cannot be changed through reassignment
// Example using const
function exampleConst() {
const z = 10;
if (true) {
const z = 20; // different variable
console.log(z); // 20
}
console.log(z); // 10
}
exampleConst();
// Uncommenting the following line will cause an error
// z = 30; // TypeError: Assignment to constant variable

Benefits of Template Literals
1.Easy Variable Insertion
Variables and expressions can be directly embedded within template literals using ${...} syntax
2.Multiline Strings
Template literals support multiline strings without needing escape characters like \n
3.Enhanced Readability
Improved readability compared to traditional string concatenation, especially for longer strings
Example Usage Multiline Strings Tagged Templates

Template literals can also be used with functions called "tagged templates". Tagged templates allow you to process template literals with a function. The function (the tag) receives the template literal's text as the first argument, and the expressions as subsequent arguments

JavaScript array

In JavaScript, an array is a data structure that allows you to store multiple values in a single variable. Arrays can hold values of different types and are ordered, meaning each value is stored at a specific, numbered position called an index. Arrays are dynamic, so you can change their size by adding or removing elements

Creating an Array

You can create an array using the array literal syntax or the Array constructor

1.Array Literal Syntax
let fruits = ["Apple", "Banana", "Cherry"];
2.Array Constructor
let fruits = new Array("Apple", "Banana", "Cherry");
Accessing Array Elements

You can access elements in an array using their index. Array indices are zero-based, meaning the first element is at index 0

Common Array Methods
1.push()
Adds one or more elements to the end of an array
fruits.push("Date");
console.log(fruits); // Output: ["Apple", "Banana", "Cherry", "Date"]
2.pop()
Removes the last element from an array
fruits.pop();
console.log(fruits); // Output: ["Apple", "Banana", "Cherry"]
3.shift()
Removes the first element from an array
fruits.shift();
console.log(fruits); // Output: ["Banana", "Cherry"]
4.unshift()
Adds one or more elements to the beginning of an array
fruits.unshift("Apricot");
console.log(fruits); // Output: ["Apricot", "Banana", "Cherry"]
5.splice()
Adds or removes elements from an array
// Remove 1 element at index 1
fruits.splice(1, 1);
console.log(fruits); // Output: ["Apricot", "Cherry"]
// Add 2 elements at index 1
fruits.splice(1, 0, "Banana", "Blueberry");
console.log(fruits); // Output: ["Apricot", "Banana", "Blueberry", "Cherry"]
6.slice()
Returns a shallow copy of a portion of an array into a new array
let citrus = fruits.slice(1, 3);
console.log(citrus); // Output: ["Banana", "Blueberry"]
7.forEach()
Executes a provided function once for each array element
fruits.forEach(function(fruit) {
console.log(fruit);
});
// Output:
// Apricot
// Banana
// Blueberry
// Cherry
8.map()
Creates a new array populated with the results of calling a provided function on every element in the calling array
let upperFruits = fruits.map(function(fruit) {
return fruit.toUpperCase();
});
console.log(upperFruits); // Output: ["APRICOT", "BANANA", "BLUEBERRY", "CHERRY"]
